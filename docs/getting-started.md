# Getting Started

Welcome to UchÅ« Docs! This project provides a simple, clean way to render Markdown documentation using PHP.

## Features

- Render Markdown files as HTML
- Automatic sidebar generation
- Simple, clean design
- Easy to extend and customize

## Quick Setup
1. Clone the repository
2. Run `composer install`
3. Configure your web server to point to the `public` directory
4. Start adding Markdown files to the `docs` directory

## Customization

You can easily customize the look and feel by modifying the CSS in the header template or adding your own styles.
